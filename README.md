# AltAug
Application to be used for Path of Exile 1. 

The purpose of this application is to provide a way for users to be able to more easily currency spam items.

Currently we are supporting:
- Crafting methods:
  - Alt/Aug
  - Alch/Scour
  - Chaos
- Supported item locations:
  - Currency tab item slot
  - Inventory (for multiple items currently only supports 1x1 items, i.e. maps)
